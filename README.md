# NwpuCampus

这是一个 HarmonyOS ArkTS 应用示例，目前主界面使用 **华为原生 Map Kit** 展示地图。

## 功能
- 主界面直接显示原生地图
- 支持缩放/拖动
- 预留定位权限（后续可接入定位）

## Map Kit 接入说明
当前项目仅完成页面与权限接入，仍需在 DevEco Studio 中完成以下配置：

1. **启用/集成华为 Map Kit SDK**（按官方文档执行）。
2. 配置 Map Kit 所需的 **App/Api Key**（按官方文档的位置填写）。
3. 完成后重新构建应用。

> 如果需要，我可以继续补上定位功能接入或 Map Kit 事件监听示例。
